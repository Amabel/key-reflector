## Usage

Replace all `'npm-package-seed'` in `package.json` with your package name in [Kebab case](https://en.wikipedia.org/wiki/Letter_case#cite_ref-28).

Also update the following fields in `package.json`:

- `name`
- `description`
- `keywords`
- `repository`
- `author`
- `homepage`

## Development

### Build

```
npm install

npm run build

npm run lint

npm run lint:fix
```
